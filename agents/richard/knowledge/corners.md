# Corners

**Last Updated**: 2026-02-19

## Summary
Corners (curate.corners.market) is an onchain curation platform on Base blockchain, backed by The Sandbox Game. Users create "corners" — topic-based link collections with their own token markets. Taste becomes tradeable.

## Status
Live and publicly accessible. Retention is the key challenge (Feb 2026).

## Key Facts
- URL: curate.corners.market
- Blockchain: Base (backed by The Sandbox / $SAND)
- Core loop: Create corner → Hold token to curate → Trade tokens → Get rewarded
- Rewards: $10k weekly to top curators/commenters via leaderboard
- Referral: 5-level system (L1: 5%, L2: 4%, L3: 3%, L4: 2%, L5: 1% trading fees)
- WoW retention: dropped 15.4% → 2.5% — focus is retention over acquisition
- North star: 10 corners >$1M market cap, $100M trading volume, 1,000 curators / 10,000 traders
- Corners API Skill: live across 31 AI skill directories

## Key Differentiators
- vs Farcaster: creates markets, not followers
- vs Zora: monetizes taste, not output
- vs Polymarket: trades ideas/culture, not binary outcomes

## Target Audiences
- Curators: internet degens, meme archeologists, Farcaster/Zora crowd
- Traders: DeFi-fluent culture speculators
- Creators: multi-platform tired of platform economics
