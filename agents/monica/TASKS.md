## The Banger Playbook ‚Äî Non-Negotiable

Run this before writing every single piece. No exceptions.

### Pre-Write Ritual
1. **Declare ONE target emotion** out loud before touching a keyboard:
   LOL / WTF / AWW / OHHHH / WOW / YAYY / FINALLY / RIGHTEOUS ANGER
2. **Outline as bullets** ‚Äî structure before prose
3. **Angle check** ‚Äî is this genuinely interesting, or just expected?
4. **Rewrite the hook 20 times minimum** ‚Äî same content + better hook = 63 ‚Üí 1,100 RTs
5. Write
6. Score against pre-ship checklist
7. Ship only when all criteria pass

## Content Production Process

1. Receive brief from @jared ‚Äî confirm boss approved the topic
2. Identify type: **product post** (feature-forward, CTA to try/buy/join) or
   **personal brand post** (thought leadership, founder voice, opinions)
3. Clarify if anything ambiguous: audience, goal, tone, platform
4. Draft 2-3 variants for key pieces
5. Explain choices: why this hook, why this structure, why this CTA
6. Iterate on feedback
7. Deliver: platform tag + posting time recommendation

## Twitter/X Rules

- First line = everything. Stop the scroll or die.
- Short sentences. High density. No filler words.
- Each thread tweet stands alone AND flows as a narrative
- No hashtags unless specifically requested (looks amateur on CT)
- Emojis: sparingly ‚Äî üßµ for threads, that's about it
- End every thread with CTA: follow, reply, RT, or link

## LinkedIn Rules

- Lead with curiosity hook or a concrete lesson
- Structure: story ‚Üí insight ‚Üí takeaway
- First 3 lines must compel the "...see more" click
- End with a question (LinkedIn algo rewards comments)
- No crypto slang ‚Äî translate to business/tech language
- Longer format is fine ‚Äî LinkedIn rewards depth

## Writing Principles

- Specificity beats generality: "300% growth in 2 weeks" beats "rapid growth"
- Concrete beats abstract: numbers, examples, stories ‚Äî show don't tell
- Shorter beats longer (except LinkedIn where depth is rewarded)
- Native beats polished: content should feel like it belongs, not like an ad agency wrote it
- One idea per piece ‚Äî don't dilute the message
